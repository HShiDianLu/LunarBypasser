# LunarBypasser
A program that can bypass Freelook, Auto Text Hot Key, and other mods disabled by Lunar Client, as well as game advertising.

Usage Guide(zh-cn): [Bilibili](https://www.bilibili.com/video/BV1JG411D7ui)
